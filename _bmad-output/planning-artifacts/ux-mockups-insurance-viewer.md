## Design System Notes

### Color Palette
- **Primary:** Deep blue (#1e3a5f) — trust, professionalism
- **Secondary:** Teal (#0d9488) — modern, financial
- **Success:** Green (#16a34a) — active status, successful parsing
- **Warning:** Amber (#d97706) — attention items
- **Error:** Red (#dc2626) — parsing errors, inactive status
- **Background:** Light gray (#f8fafc)
- **Cards:** White (#ffffff) with subtle shadow

### Typography
- **Hebrew Font:** "Assistant" (Google Fonts) or "Heebo" — clean, modern Hebrew typeface
- **Numbers:** Tabular lining for financial data alignment
- **Headings:** Bold, 18-24px
- **Body:** Regular, 14-16px
- **Financial figures:** Semi-bold, slightly larger for emphasis

### Layout Principles
- Full RTL with `dir="rtl"` and `lang="he"`
- Max content width: 1200px centered
- Card-based layout with 16-24px padding
- Responsive grid: 3 columns desktop → 2 tablet → 1 mobile
- Navigation tabs persistent at top
- Back button always visible on drill-down screens

### Interaction Patterns
- File upload: drag-and-drop + click to browse
- Dashboard cards: hover effect → click to drill down
- Tables: sortable columns, hover highlight
- Charts: simple bar/pie charts for distribution views
- No modals — all content inline
- No infinite scroll — paginate if needed

### Accessibility
- WCAG 2.1 AA minimum
- Color contrast ratios: 4.5:1 for text
- Focus indicators for keyboard navigation
- Screen reader labels for charts and data visualizations
- No color-only indicators (always include text/icon)
